# gcloud-in-go image

Use this image when you want to use `go` and `gcloud` in the same job

## contents

- base:
  - golang:1.19
- directories:
  - `/workspace` default working dir for `run` commands
- languages:
  - `python`
- tools:
  - `gcloud` installed via rapid channel, components include:
    - `alpha`
    - `beta`
    - `kubectl`
  - `rsync`
  - `wget`
